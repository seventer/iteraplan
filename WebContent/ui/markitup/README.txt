markItUp! universal markup jQuery editor

markItUp! is a JavaScript plugin built on the jQuery library.
It allows you to turn any textarea into a markup editor.
Html, Textile, Wiki Syntax, Markdown, BBcode or even your own Markup system can be easily implemented.

http://markitup.jaysalvat.com/home/

-----------------------------------------------------------

markItUp wird als RichEditor für die Wiki integration verwendet.

Um die XWiki integration zu gewährleisten, wurde lediglich die Datei "set.js" unter "sets/wiki/" angepasst.
In dieser Datei können ohne Probleme auch weitere XWiki Tags ergänzt werden
bzw. auch eigene erstellt werden.
