A model used to interface between the domain-specific code and the graphical export of matrices.

This package contains classes that can be used to store all information necessary for a landscape export. This is done in a way that is independent of the domain model used in iteraplan, thus decoupling the graphic export from the rest of the application. Outgoing dependencies of this package are limited to utility classes.

The main entry point is the {@link de.iteratec.iteraplan.businesslogic.exchange.common.landscape.beans.LandscapeDiagram} class.
