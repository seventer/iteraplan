This package contains the logic for creating the different visio exports.

Overview

The following classes are responsible for creating the different visio diagram types:

  • {@link de.iteratec.iteraplan.businesslogic.exchange.visio.VisioBubbleExport}
    • creates the Portfolio Diagram.
  • {@link de.iteratec.iteraplan.businesslogic.exchange.visio.informationflow.VisioInformationFlowExport}
    • creates the Information Flow Diagram.
  • {@link de.iteratec.iteraplan.businesslogic.exchange.visio.VisioMasterplanExport}
    • creates the Masterplan Diagram.
  • {@link de.iteratec.iteraplan.businesslogic.exchange.visio.landscape.VisioLandscapeDiagramExport}
    • creates the Landscape Diagram.
