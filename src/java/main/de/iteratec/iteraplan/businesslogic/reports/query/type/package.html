A meta model for the domain model used primarily for constructing user defined queries.

For each BuildingBlock a Type exists that describes its attributes and its relations to other BuildingBlocks by an Extension. The types are used by the presentation tier to generate query forms for the different building blocks. The types are also used when passing a user defined query to the business logic.
