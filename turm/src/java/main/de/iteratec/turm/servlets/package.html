This package contains the servlets, which in turn contain most of the logic.
  • {@link de.iteratec.turm.servlets.PasswordServlet}
    • Responsible for changing a users password. Access to this class should not be restriced to logged in users.
  • {@link de.iteratec.turm.servlets.RoleServlet}
    • Responsible for CRUD operations on roles and the assignement from roles to users.
  • {@link de.iteratec.turm.servlets.UserServlet}
    • Responsible for CRUD operations on users and the assignement from users to roles.
  • {@link de.iteratec.turm.servlets.TurmServlet}
    • Superclass of all servlets. Contains common logic needed by all servlets.
